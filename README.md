# LawyerUp – Legal Management App

**Language:** C++  
**Project Type:** Console-based Application  
**Developer:** Yaseen Dawood – BSc IT Student, North-West University  
**GitHub:** [github.com/YDawood03/LawyerUp](https://github.com/YDawood03/LawyerUp)

---

## 📖 Overview

LawyerUp was my **first-year university project** – my very first attempt at building something real with code. Fresh into the world of programming, I wanted to create an application that wasn't just another academic exercise, but something that could actually be **useful in the real world**.

Having worked at a law firm, I saw firsthand how legal professionals manage clients, track invoices, and juggle court dates. I thought: *"What if I could build something to help with that?"*

This console-based legal management system was my answer. It manages clients, invoices, court dates, and even generates basic legal documents. Was it perfect? Not at all. But it was my first real step toward understanding how code can solve real problems.

---

## 🚀 Features

| Feature | Description |
| :--- | :--- |
| **Client Management** | Add, remove, and display client information (name, reference, phone, email) |
| **Invoice Generation** | Track work done, prices, and payments; display total amounts due |
| **Court Dates** | Add, remove, and view court dates; saved to a text file for record keeping |
| **Document Generation** | Create legal documents like Notice of Withdrawal, Letter of Demand, and ANC Contract |
| **Secure Login** | Restricted access with usernames and passwords |

---

## 📝 What I Learned

Building LawyerUp taught me that programming isn't just about syntax – it's about **solving problems**. I learned:

- How to structure code for readability and function
- The importance of planning before coding
- That users need clear, simple interfaces (even in a console app)
- How to persist data using files
- That debugging is 50% of the work (and 100% worth it)
